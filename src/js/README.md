# Carpeta JavaScript

Esta carpeta contiene los archivos JavaScript principales de la aplicación.

## Estructura

- **main.js** - Punto de entrada de la aplicación Vue 3
  - Importa la aplicación principal (App.vue)
  - Carga los estilos CSS globales
  - Monta la aplicación en el elemento #app

## Módulos importados

- `../App.vue` - Componente raíz de la aplicación
- `../styles/style.css` - Estilos globales

## Cómo agregar nuevo código JavaScript

1. Crea un nuevo archivo `.js` en esta carpeta
2. Importa en `main.js` si necesita ejecución global
3. O importa en el componente Vue donde lo necesites
